# Style Vocabulary Reference

A curated list of style descriptors that reliably influence AI media generation outputs.

## Photography Styles

| Term | Effect |
|------|--------|
| `35mm film grain` | Organic texture, vintage feel |
| `medium format` | High detail, shallow DOF rendition |
| `daguerreotype` | Antique, sepia-toned, oval vignette |
| `cyanotype` | Blue-tinted, blueprint aesthetic |
| `double exposure` | Two images merged, ethereal |
| `long exposure` | Motion blur, light trails |
| `infrared photography` | White foliage, dark skies |
| `macro photography` | Extreme close-up, abstract detail |

## Cinematic References

| Term | Effect |
|------|--------|
| `anamorphic lens flare` | Horizontal bokeh, cinematic streak |
| `Kodak Vision3` | Warm shadows, filmic color |
| `Roger Deakins cinematography` | Naturalistic, golden light |
| `Wong Kar-wai aesthetic` | Saturated, motion blur, intimate |
| `Kubrick symmetry` | One-point perspective, cold precision |
| `Terrence Malick` | Lens flare, natural light, lyrical |

## Illustration / Art Styles

| Term | Effect |
|------|--------|
| `editorial illustration` | Bold, graphic, editorial magazine feel |
| `risograph print` | Layered, slightly misregistered colors |
| `screen printing` | Flat bold colors, visible texture |
| `woodblock print` | Ukiyo-e influenced, line-dominant |
| `gouache illustration` | Matte, opaque, hand-painted |
| `linocut` | High contrast, carved texture |

## 3D / Digital

| Term | Effect |
|------|--------|
| `octane render` | Photorealistic, ray-traced, rich materials |
| `Blender clay render` | Grey matte, form-focused |
| `low poly` | Geometric facets, minimal |
| `voxel art` | Minecraft-like blocks |
| `cel-shaded` | Flat fills with hard outline (anime/cartoon) |
| `unreal engine 5 render` | Hyperrealistic real-time render quality |

## Lighting Vocabulary

| Term | Effect |
|------|--------|
| `golden hour` | Warm, directional, long shadows |
| `blue hour` | Cool, twilight, low contrast |
| `chiaroscuro` | Dramatic dark/light contrast |
| `Rembrandt lighting` | Triangle of light under eye |
| `split lighting` | Exactly half face lit |
| `rim lighting` | Light behind subject, glowing edge |
| `practical lights only` | Only in-scene lights (lamps, screens) |
| `motivated light` | Light sourced from visible in-scene element |
| `overcast diffused` | Even, shadowless, soft |
| `harsh midday sun` | Unflattering, high contrast, hard shadows |

## Mood / Atmosphere Descriptors

| Term | Effect |
|------|--------|
| `liminal space` | Empty, transitional, slightly uncanny |
| `vaporwave` | Pink/teal, retro-futuristic, nostalgic |
| `dark academia` | Moody, library-set, intellectually gothic |
| `cottagecore` | Pastoral, warm, hand-made aesthetic |
| `solarpunk` | Green, utopian, hopeful future |
| `brutalist` | Raw concrete, functional, imposing |
| `wabi-sabi` | Imperfection, transience, quiet |
| `maximalist` | Dense, layered, every surface decorated |
