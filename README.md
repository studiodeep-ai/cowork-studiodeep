# StudioDeep Claude Plugins

Personal Claude Code plugin marketplace for StudioDeep projects.

## Plugins

- **media-prompter** — Generates rich, structured prompts for AI media generation tools (Midjourney, DALL-E, Stable Diffusion, Sora, RunwayML)

## Install

```bash
claude plugin marketplace add /Users/timdpaep/LocalDev/studiodeep/claude-plugins --scope user
claude plugin install media-prompter
```
